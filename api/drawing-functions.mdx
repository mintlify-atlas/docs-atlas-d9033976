## Overview

FLTK provides a comprehensive set of portable drawing functions for creating graphics, rendering text, drawing shapes, and manipulating images. These functions work across all supported platforms (Windows, macOS, Linux/X11).

**Header:** `<FL/fl_draw.H>`

## Color Functions

<ParamField path="fl_color" type="void fl_color(Fl_Color c)">
  Sets the color for all subsequent drawing operations.
  
  <ParamField path="c" type="Fl_Color">
    Color index from the FLTK color map
  </ParamField>
  
  For color-mapped displays, a color cell is allocated from fl_colormap the first time a color is used.
</ParamField>

```cpp
void fl_color(Fl_Color c);
void fl_color(int c);  // For backward compatibility
```

<ParamField path="fl_color" type="void fl_color(uchar r, uchar g, uchar b)">
  Sets the color using RGB components.
  
  <ParamField path="r" type="uchar">
    Red component (0-255)
  </ParamField>
  
  <ParamField path="g" type="uchar">
    Green component (0-255)
  </ParamField>
  
  <ParamField path="b" type="uchar">
    Blue component (0-255)
  </ParamField>
  
  The closest possible match to the RGB color is used on color-mapped displays.
</ParamField>

```cpp
void fl_color(uchar r, uchar g, uchar b);
```

<ParamField path="fl_color" type="Fl_Color fl_color()">
  Returns the last fl_color() that was set. Can be used for state save/restore.
</ParamField>

## Line Style

<ParamField path="fl_line_style" type="void fl_line_style(int style, int width = 0, char* dashes = 0)">
  Sets how to draw lines (the "pen").
  
  <ParamField path="style" type="int">
    Bitmask of line style, cap style, and join style:
    - `FL_SOLID` - Solid line (0)
    - `FL_DASH` - Dashed line (1)
    - `FL_DOT` - Dotted line (2)
    - `FL_DASHDOT` - Dash-dot pattern (3)
    - `FL_DASHDOTDOT` - Dash-dot-dot pattern (4)
    - `FL_CAP_FLAT` - Flat end caps (0x100)
    - `FL_CAP_ROUND` - Round end caps (0x200)
    - `FL_CAP_SQUARE` - Square end caps (0x300)
    - `FL_JOIN_MITER` - Miter join (0x1000)
    - `FL_JOIN_ROUND` - Round join (0x2000)
    - `FL_JOIN_BEVEL` - Bevel join (0x3000)
    - `FL_UNIFORM_WIDTH` - Uniform width in scaled contexts (0x10000)
  </ParamField>
  
  <ParamField path="width" type="int" default="0">
    Line thickness in pixels (0 = system default)
  </ParamField>
  
  <ParamField path="dashes" type="char*" default="0">
    Array of dash lengths in pixels, terminated with 0
  </ParamField>
  
  <Warning>Reset line style to default with `fl_line_style(0)` when done, or box/frame drawing will fail.</Warning>
  <Warning>On Windows, set line style AFTER setting color, or style settings will be lost.</Warning>
</ParamField>

<ParamField path="fl_antialias" type="void fl_antialias(int state)">
  Turns antialiased line drawing ON or OFF (Windows only).
</ParamField>

<ParamField path="fl_antialias" type="int fl_antialias()">
  Returns whether line drawings are currently antialiased.
</ParamField>

## Example Usage

### Drawing Basic Shapes

```cpp
void MyWidget::draw() {
    // Set color and draw filled rectangle
    fl_color(FL_RED);
    fl_rectf(10, 10, 100, 50);
    
    // Draw rectangle outline
    fl_color(FL_BLACK);
    fl_rect(10, 10, 100, 50);
    
    // Draw circle
    fl_color(FL_BLUE);
    fl_begin_polygon();
    fl_circle(150, 35, 25);
    fl_end_polygon();
    
    // Draw arc
    fl_color(FL_GREEN);
    fl_arc(200, 10, 50, 50, 0, 270);
}
```

### Drawing with Line Styles

```cpp
// Dashed line
fl_color(FL_BLACK);
fl_line_style(FL_DASH, 2);
fl_line(10, 10, 200, 10);

// Custom dash pattern
char dashes[] = {10, 5, 2, 5, 0}; // 10 on, 5 off, 2 on, 5 off
fl_line_style(FL_DASH, 1, dashes);
fl_line(10, 20, 200, 20);

// Reset to solid
fl_line_style(0);
```

### Drawing Text

```cpp
// Simple text
fl_font(FL_HELVETICA, 14);
fl_color(FL_BLACK);
fl_draw("Hello, FLTK!", 10, 30);

// Rotated text
fl_draw(90, "Vertical Text", 50, 100);

// Formatted text in box
fl_draw("Centered\nMulti-line\nText",
        10, 50, 200, 100,
        FL_ALIGN_CENTER);
```

### Complex Polygon

```cpp
// Draw a star
fl_color(FL_YELLOW);
fl_begin_polygon();
for (int i = 0; i < 10; i++) {
    double angle = i * 36 * M_PI / 180;
    double r = (i % 2 == 0) ? 50 : 20;
    fl_vertex(100 + r * cos(angle),
              100 + r * sin(angle));
}
fl_end_polygon();
```

### Using Transformations

```cpp
fl_push_matrix();
fl_translate(100, 100);
fl_rotate(45);
fl_scale(2.0);

fl_color(FL_RED);
fl_rectf(-25, -25, 50, 50);

fl_pop_matrix();
```

### Drawing Raw Image Data

```cpp
// Create RGB image data
uchar* img = new uchar[100 * 100 * 3];
for (int i = 0; i < 100 * 100 * 3; i += 3) {
    img[i]   = 255;  // R
    img[i+1] = 0;    // G
    img[i+2] = 0;    // B
}

// Draw it
fl_draw_image(img, 10, 10, 100, 100, 3);
delete[] img;
```
